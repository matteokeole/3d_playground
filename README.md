# 3D WebGL playground

### Setup

```bash
git clone git@github.com:matteokeole/webgl_playground --single-branch --recurse-submodules --remote-submodules --shallow-submodules
```

### Controls

| Action | Key |
| --- | --- |
| Walk forward | `W` |
| Walk backward | `S` |
| Strafe left | `A` |
| Strafe right | `D` |
| Fly up | `Space` |
| Fly down | `LeftCtrl` |
